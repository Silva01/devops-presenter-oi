O Que é o Puppet ?

Puppet é uma ferramenta e plataforma que ajuda na gerência de configuração de servidores e serviços 


							  
						define nginx::vhost (
							$port = '80',
							$docroot = '/var/www',
							$servername = $title,
						){

						file {'/etc/nginx/conf.d/teste.conf':
							ensure  => file,
							group   => 'root',
							mode    => '0644',
							content => epp('nginx/vhost.conf.epp', 
							{ 'docroot' => $docroot, 'port'    => $port}),
							}
						}

Quais suas Caracteristicas ?

Linguagem de configuração declarativa. na qual você define as configurações desejada do sistema

Arquitetura cliente/servidor para distribuição de configuração

Simula as mudanças de configuração sem realizar modificações no host gerenciado.

Extensível por meio de módulos e plugins por conta do Ruby

Histórico a cada nova execução do agente puppet

Possui suporte aos principais Sistemas operacionais de mercado
